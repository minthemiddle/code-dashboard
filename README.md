# About

*CodeDashboard* is the attendant management system for multi-days events, especially for coding camps.

# Installation

## Mac

- `brew install go`
- `brew install mongodb`

## All platforms

- Fork project
- `git clone git@github.com:CodeDesignInitiative/code-dashboard.git`
- `cd code-dashboard`
- `go run *.go`
- Profit
